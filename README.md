# ⚛️ React Codes

A collection of React mini-projects and components built while learning React.js — covering hooks, forms, state management, API calls, and interactive UIs.

---

## 🗂️ Projects Overview

| Component | Description |
|---|---|
| 🏧 `ATM-machine.jsx` | Simulates ATM operations like deposit, withdraw & balance check |
| 📚 `App.jsx` | Book tracker to monitor reading progress across pages |
| 💬 `FeedbackForm.jsx` | Feedback form with controlled inputs |
| 🔐 `PasswordChecker.jsx` | Password input with show/hide toggle |
| 📝 `Registration Form.jsx` | User registration form with state management |
| 🛒 `Shopping-cart.jsx` | Shopping cart with product listing, add/remove items |
| 🎓 `Student-Portal.jsx` | Student portal to add and manage student records |
| 💰 `TipCalculator.jsx` | Calculates tip amount based on bill and percentage |
| 🚦 `Traffic-LightApp.jsx` | Interactive traffic light simulator |
| ⌨️ `TypingSpeedTester.jsx` | Measures typing speed with a live timer |
| 🏛️ `UniversityApplicationForm.jsx` | Multi-step university application form |
| 👤 `UserProfile.jsx` | Fetches and displays user profile data from an API |
| 🌦️ `WeatherCard.jsx` | Live weather lookup by city name |
| 📊 `live-dashboard.jsx` | Real-time study session tracker with live clock |
| 🔤 `timeCast.jsx` | Simple controlled text input component |

---

## Concepts Practiced

- `useState` — Local state management across all components
- `useEffect` — Side effects, API calls, timers & real-time updates
- Controlled Forms — Handling inputs, validation & multi-step forms
- Conditional Rendering — UI changes based on state
- List Rendering — Dynamic product/student lists with keys
- Event Handling — Buttons, inputs, and form submissions
- API Integration — Fetching live data (weather, user profiles)

---

## Tech Stack

- **React 19**
- **Vite** (dev server & bundler)
- **JavaScript (ES6+)**

---

## How to Run

```bash
# Clone the repo
git clone https://github.com/RadhikaKapoor383/React-Codes.git

# Navigate into the folder
cd React-Codes

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Then open `http://localhost:5173` in your browser.

> To switch between components, update the import in `main.jsx`.
